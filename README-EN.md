# 介绍

 Asp.Net Core Webapi 项目快速启动模板

# 项目结构

- `Congfiguration`:项目配置库
- `Core`:核心公共基础库
- `Utility`:工具类库
- `Filter`:过滤器库
- `Models`:实体层
- `Repository`,`IRepository`:仓储层
- `IServices`,`Services`:服务层
- `Webapi`: Webapi层

# 技术栈

- 序列化：`Newtonsoft.Json`
- 身份验证：`Jwt.Net`
- 数据库ORM：`SqlSugar`
- 接口缓存：`CacheOutput`
- 实体映射：`AutoMapper`
